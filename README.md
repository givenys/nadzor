# Face Recognition System

Система распознавания лиц на основе InsightFace с адаптивным улучшением освещённости.

## 🚀 Возможности

- **Распознавание лиц в реальном времени** с использованием InsightFace (ArcFace)
- **Адаптивное улучшение освещённости** (CLAHE + gamma-коррекция) для работы в полумраке
- **Векторизованное сравнение** эмбеддингов для высокой производительности
- **Панель управления** с настройкой параметров в реальном времени
- **Модульная архитектура** для лёгкого расширения

## 🛠 Установка

1. **Установите зависимости:**
```bash
pip install -r requirements.txt
```

2. **Добавьте эталонные лица:**
   - Поместите фотографии в папку `data/reference_faces/`
   - Имя файла будет именем человека (без расширения)
   - Пример: `ivan.jpg` → "ivan"

## 🎮 Запуск

```bash
python main.py
```

### Управление

| Клавиша | Действие |
|---------|----------|
| `Q` | Выход |
| Трекбары | Настройка параметров в реальном времени |

### Параметры панели управления

- **Threshold x100** — Порог сходства (0.3-0.6 типично)
- **Frame Skip** — Пропуск кадров для производительности
- **Min Face Area** — Минимальная площадь лица
- **CLAHE Clip** — Контраст для улучшения освещённости
- **Gamma Dark x10** — Осветление тёмных зон
- **Brightness Thresh** — Порог активации улучшения
- **[RELOAD BASE]** — Перезагрузка базы лиц

## 🔧 Настройка под ваши условия

### Для низкой освещённости:
1. Увеличьте `Gamma Dark` до 2.0-2.2
2. Увеличьте `CLAHE Clip` до 2.5-3.0
3. Снизьте `Brightness Thresh` до 50-60

### Для уменьшения ложных срабатываний:
1. Снизьте `CLAHE Clip` до 1.5
2. Увеличьте `Threshold` до 0.55-0.6
3. Увеличьте `Min Face Area`
